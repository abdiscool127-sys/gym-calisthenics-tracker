# Wat doen de talen?

## Heel kort:
- JavaScript = pagina reageert.
- Vue.js = pagina ziet er netjes uit.
- Python en Flask = server en data.
- Git en GitHub = je werk opslaan.

## JavaScript
JavaScript zorgt dat de pagina kan reageren op wat je doet. Als je op een knop klikt, gebeurt er iets.

## Vue.js
Vue.js maakt de pagina netjes en overzichtelijk. Het zorgt dat dingen automatisch veranderen.

## Python
Python is de taal van de server. De server bewaart de trainingsgegevens.

## Flask
Flask is het programma dat de server draait. Het zegt tegen de website: "Hier is je data!"

## Git
Git bewaart je wijzigingen. Als je een fout maakt, kun je alles terugrollen.

## GitHub
GitHub is online. Daar kunnen anderen (en je docent) zien wat je hebt gedaan.

## Hoe werkt het samen?
1. Website = wat je ziet.
2. Server = wat achter het scherm gebeurt.
3. Git/GitHub = hoe je het hebt bijgehouden.
