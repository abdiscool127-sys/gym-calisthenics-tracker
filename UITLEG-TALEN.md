# Wat doen de talen?

Dit is een korte, makkelijke uitleg die je aan iemand zonder technische kennis kunt voorlezen.

## Heel kort (eenvoudig)
- JavaScript: zorgt dat de pagina reageert op klikken en invullen.
- Vue.js: helpt om de pagina overzichtelijk te maken (structuur op de pagina).
- Python: is de taal voor de server (het programma dat op je computer draait en data geeft).
- Flask: is het kleine programma dat de server laat draaien (maakt het makkelijk om web-antwoorden te geven).
- Git: houdt bij welke wijzigingen je hebt gemaakt (als een geschiedenisboek).
- GitHub: is de plek online waar je die geschiedenis (je project) laat zien.

## Iets uitgebreider (in gewone taal)

- JavaScript: denk aan de logica die zegt "als iemand op deze knop klikt, laat dan deze tekst zien".
- Vue.js: organiseert stukken van de pagina zodat ze makkelijk opnieuw gebruikt en aangepast kunnen worden.
- Python: schrijft de instructies voor de server — wat moet de server doen als iemand iets vraagt.
- Flask: het gereedschap waarmee die Python-instructies als een eenvoudige website worden aangeboden.
- Git: vergelijkbaar met "opslaan met een notitie waarom je hebt veranderd". Je kunt terug naar eerdere versies.
- GitHub: een online map met je project waar anderen kunnen kijken.

## Hoe werken ze samen? (kort verhaal)
1. De gebruiker opent de pagina (frontend).
2. De pagina vraagt: "Wat zijn de doelen?" aan de server.
3. De server (Python + Flask) antwoordt met de gegevens.
4. De pagina laat die gegevens zien en de gebruiker kan iets toevoegen.
5. Als je wilt, gebruik je Git/GitHub om te laten zien welke stappen je hebt genomen.
